# Function-Image
C++ Easyx绘制函数图像

本程序适用于二次函数，一次函数，反比例函数的图像绘制

使用时会弹出两个小黑窗，
从大的小黑窗里输入，输入完所有东西后再按任意一个键就关闭。

然后再看小的小黑窗（实在是太不方便了）

目前只能一次性使用（搞完了就关上）
